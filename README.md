# MonBazouMP
# https://monbazou.online/
This multiplayer was not created by the developers of the original game, but is a mod created by one enthusiast(By me). The mod is free, but it still has some bugs. At the moment, the game is synchronized:
* 🟢cars
* 🟢installation of parts
* 🟢shopping in the store
* 🟢weather and time
* 🟢as well as the position of objects

#?
* This mod is paired with the Revolution Core - Modding API, and installation is required.
* Revolution Core - Modding API on Nexusmods - https://www.nexusmods.com/monbazou/mods/420
* Revolution Core - Modding API on GitHub - https://github.com/Revolution2577/Mon-Bazou-Modding-API
* Mon Bazou MP on Nexusmods - https://www.nexusmods.com/monbazou/mods/378
